## Video Editing Recommendations

Open with the portfolio homepage, not a logo-only intro. Clients should immediately see that this is a real portfolio with working project examples.

Group the projects by client problem instead of showing random screenshots. A good order is: workflow systems, business operations, AI applications, mobile/tracking apps, then contact CTA.

Keep each project screenshot on screen for about 1-2 seconds. Add one short label per screenshot, such as "AI Forecasting," "LGU Reporting," "Delivery Dashboard," or "Mobile Voting App."

Use zoom-ins on dashboards, cards, charts, maps, detection boxes, and QR tracking sections. These details quickly communicate technical capability without needing long explanations.

End with a clear call to action: "Need a custom system? Message NeuralDev today." Show the portfolio link and contact page in the final frame.
