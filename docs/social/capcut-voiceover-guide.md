# CapCut Guide for NeuralDev Facebook Promo Video

Use this guide to add voiceover, music, and final polish to the portfolio promo video.

## Files to Use

- Video: `media/social/neuraldev-facebook-portfolio-promo.mp4`
- Cover image: `media/social/neuraldev-facebook-portfolio-cover.png`
- Caption/script pack: `docs/social/facebook-portfolio-promo.md`

## Recommended CapCut Setup

- Format: 9:16 vertical
- Resolution: 1080x1920
- Length: 26 seconds
- Export format: MP4
- Best for: Facebook Reels, Facebook Stories, and Facebook feed video

## Step-by-Step Instructions

### 1. Open CapCut

Use either:

- CapCut mobile app
- CapCut desktop app
- CapCut online editor at `https://www.capcut.com`

Create a new project.

### 2. Import the Promo Video

Upload this file:

`media/social/neuraldev-facebook-portfolio-promo.mp4`

Place it on the main video timeline.

Make sure the project canvas is set to vertical or 9:16.

### 3. Add Voiceover

You can choose either option:

#### Option A: Record Your Own Voice

1. Tap or click `Audio`.
2. Choose `Voiceover`.
3. Play the video and record while reading the script below.
4. Keep your voice clear, confident, and natural.

#### Option B: Use Text-to-Speech

1. Tap or click `Text`.
2. Add text from the voiceover script.
3. Select `Text-to-Speech`.
4. Choose a professional voice.
5. Adjust the timing so each line matches the scene.

### 0-2 seconds

Looking for a developer who builds practical digital solutions?

### 2-5 seconds

## Add Background Music

1. Go to `Audio`.
2. Choose a clean technology, corporate, or upbeat background track.
3. Set music volume low, around 8-15%.
4. Keep voiceover volume higher, around 90-100%.

The voice should be easy to hear. The music should only support the video.

## Add Captions

If you recorded your own voice:

1. Use `Auto Captions`.
2. Choose English.
3. Review the generated captions.
4. Correct any wrong words.

Suggested caption style:

- White text
- Medium size
- Simple font
- No heavy effects
- Place captions near the lower-middle area, but avoid covering the CTA at the bottom

## Optional Text Overlays

If you want to add extra labels, use short text only:

- Custom Web Systems
- AI Applications
- Mobile Apps
- Management Systems
- QR Tracking
- Dashboards
- Built for Real-World Use

Do not place too many overlays. The video already has main text, so keep extra labels minimal.

## Final Export Settings

Use these export settings:

- Resolution: 1080p
- Frame rate: 30 fps
- Format: MP4
- Quality: High

After exporting, upload the video to Facebook Reels or your Facebook page.

## Facebook Upload Steps

1. Create a new Facebook post or Reel.
2. Upload the exported CapCut video.
3. Paste the caption from `docs/social/facebook-portfolio-promo.md`.
4. Replace `[insert portfolio link]` with your real website link.
5. Add the cover image if Facebook lets you choose a custom thumbnail.
6. Publish or schedule the post.
